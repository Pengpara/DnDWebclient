📜 Before You Begin

This demo uses the Mistral AI API to power your dungeon master. You'll need your own (free) API key to play.

Don't worry! It's quick and free to get started.

🗝️ How to Get Your API Key:

Go to console.mistral.ai
Create a free account (if you don't have one)
Navigate to "API Keys" in the sidebar
Click "Create new key"
Copy your API key and paste it below
* Your key is stored only in your browser and never sent to my server
